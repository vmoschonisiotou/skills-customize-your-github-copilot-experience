

# 📘 Assignment: Hangman Game

## 🎯 Objective

Build the classic Hangman word-guessing game using Python. Practice string manipulation, loops, conditionals, and random selection.

---

## 📝 Tasks

### 🛠️ Task 1: Set Up the Game

#### Description
Initialize your game by preparing a list of possible words and setting up the main variables to track the game state.

#### Requirements
Completed program should:
- Contain a list of at least 5 possible words
- Randomly select a secret word from the list
- Initialize variables for guessed letters and incorrect attempts

---

### 🛠️ Task 2: Implement the Game Loop

#### Description
Create the main loop where the player guesses letters, and the game updates and displays progress.

#### Requirements
Completed program should:
- Prompt the player to guess a letter each turn
- Show the current progress (e.g., _ a _ _ m a n)
- Track and display the number of incorrect guesses remaining
- End the game when the word is guessed or attempts run out
- Display a win or lose message at the end

---

**Encouragement:**
Have fun and experiment! Try adding extra features like tracking guessed letters or letting the player choose the number of attempts.
